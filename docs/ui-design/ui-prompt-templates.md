```

3. 推荐大师展示区域
   - 横向滚动的大师卡片列表
   - 大师头像、姓名、领域、评分、价格
   - 在线状态和认证标识

4. 推荐内容区域
   - 课程卡片和社群动态
   - 基于用户兴趣的个性化推荐

- Element Plus UI 组件库
```
